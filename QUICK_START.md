## â“ Troubleshooting

### "Module not found" error
**Solution:** Run `pip install -r requirements.txt`

### "heart_model.pkl not found"
**Solution:** Run `python train_model.py` first

### App shows "No Disease" for all inputs
**Solution:** The model files might be corrupted. Delete `.pkl` files and rerun `python train_model.py`

### Port 8501 already in use
**Solution:** Run `streamlit run streamlit_app.py --server.port 8502`
